Get the starting date and time of the sporting events.

== Description ==

Get the starting date and time of the sporting events. Currently you can get events from NBA, English, Spanish, German, Italian and French football leagues and cups.

== Installation ==

1. Download plugin from <a href='http://www.whattimeisthe.com/files/wp-plugin.zip'>here</a>.
2. Upload whattimeisthe folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the plugin to the sidebar.
